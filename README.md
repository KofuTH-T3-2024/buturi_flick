# 作ろうとしたきっかけ
物理フリックキーボードを作ろうと思ったきっかけは学園祭でした。学園祭ではクラス作品を展示していて、そこで学校の金で物理フリックキーボードを作れるのではと思い、制作をしました。

# 制作
制作は主にgoogle japanが公開しているgithubを参考にして作りました。\
